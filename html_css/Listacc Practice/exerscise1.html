HTML5 ELEMENTS

  1. Footer element: the section located at the bottom of the web page.
  2. Nav: this is a tag that represent a section of the page that link to other pages. This is a section with the navigate link.
  3. Acticle element replace the div element. it can act as a container.
  4. Audio element is used to embedd audio in a web page.
  5. Progrss element provide the ability to create progress bar in a web page.

It is advisable to start from html if you want to learn web development. HTM is a standard that has been set by W3Schools.
